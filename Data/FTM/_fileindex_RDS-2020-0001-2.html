Fire and tree mortality database (FTM) (2nd Edition)

Research Data Publication File Index

File Folder Description
_metadata_RDS-2020-0001-2.html   Metadata file in HTML format containing a description of the content, quality, and other characteristics of the data.
_metadata_RDS-2020-0001-2.xml   Metadata file in Extensible Markup Language (XML) format containing a description of the content, quality, and other characteristics of the data.
Dataset_citations.csv \Data Comma-separated values (CSV) file containing the main citation for each contributed dataset in the Fire and Tree Mortality (FTM) database for additional information about the data collected.
Dataset_primary_contacts.csv \Data CSV file containing dataset names as they appear in the FTM database and the associated primary contact information.
FTM_fires.csv \Data CSV file containing fire names, year, dataset contact, and location for fires in the FTM database.
FTM_trees.csv \Data CSV file contains tree-level records of fire injury, tree size, and bark beetle attacks. The database is a compilation of 40 individual datasets, which are identified for each record.
Species_BarkThickness.csv \Data CSV file containing the list of species found in the FTM database and the bark thickness information used to evaluate FOFEM version 6.4 model accuracy.
FTM_Fulldatabase_firelocations.tif \Supplements Tagged Image Format file containing a map showing the geographic location of fires included in the FTM database by year of fire occurrence.
